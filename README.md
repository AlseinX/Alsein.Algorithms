# Alsein.Algorithms

## Targets Supported

| .NET Implemention | Minimum Version |
| ------------- | :-----: |
| .NET Standard | 2.0 |
| .NET Core | 2.0 |
| .NET Framework | 4.5 |

## Add To Your Projoct

Run the following command within your project directory.

    dotnet add package Alsein.Algorithms
